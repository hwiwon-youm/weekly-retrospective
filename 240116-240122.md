# 기술 리포트
### 1. getattr
```
def getattr(object, name, default=None):
      pass
```
object: 객체  
name: 문자열  
default: 속성이 없는 경우 반환값

객체의 명명된 속성 값/메서드를 반환한다. name은 문자열, 문자열이 객체의 속성 중 하나의 이름인 경우 결과는 해당 속성의 값이다.
```
getattr(x, 'foobar')
>> x.foobar
```
